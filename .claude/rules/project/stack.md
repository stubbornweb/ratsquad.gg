## Stack-Specific Rules

### Next.js App Router Patterns
- **Always** use App Router (`app/` directory) ‚Äî never Pages Router
- **Always** use `next/font/google` for Google Fonts ‚Äî never CDN links
- **Always** use `next/link` for internal navigation ‚Äî never `<a>` tags
- **Always** use `next/image` for optimized images when applicable
- **Never** expose API secrets client-side ‚Äî use API routes for external calls
- **Always** add explicit `"use client"` directive when using hooks or browser APIs

### Tailwind CSS v4 Patterns
- **Always** use `@import "tailwindcss"` syntax (v4) ‚Äî not `@tailwind` directives
- **Always** define custom theme values in `@theme inline` block in `globals.css`
- **Always** use CSS custom properties (`:root` vars) for design tokens
- **Current color palette:**
  - `--bg-main`: `#09090B` ‚Äî Primary background
  - `--bg-alt`: `#0D0D0D` ‚Äî Secondary background
  - `--bg-footer`: `#050505` ‚Äî Deepest background
  - `--bg-card`: `#111111` ‚Äî Card background
  - `--accent`: `#FFD700` ‚Äî Cyber/Tactical Yellow (brand accent)
  - `--text-main`: `#FAFAFA` ‚Äî Primary text
  - `--text-muted`: `#A1A1AA` ‚Äî Secondary text
  - `--text-dark`: `#71717A` ‚Äî Tertiary text
  - `--border-color`: `#18181B` ‚Äî Primary borders
  - `--border-subtle`: `#27272A` ‚Äî Subtle borders
  - `--border-accent`: `rgba(255, 215, 0, 0.3)` ‚Äî Accent borders
- **Always** use `clamp()` for responsive typography
- **Never** use rounded corners on buttons ‚Äî `border-radius: 0` is a strict design rule
