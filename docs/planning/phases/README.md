# Project Phases

| Phase | Name | Stories | Priority | Status |
|-------|------|---------|----------|--------|
| 1 | Content Completion | 3 | critical | Complete |
| 2 | Visual Polish | 3 | high | Complete |
| 3 | Performance | 1 | medium | Partial |
| 4 | Code Quality | 1 | low | Not Started |

Phases are ordered by owner priority:
1. Finish remaining sections/pages
2. Visual polish (animations, transitions, micro-interactions)
3. Performance (loading speed, optimization)
4. Code quality (extract components, reduce CSS, clean architecture)

Run `/ww:plan` to start working on the current phase.
